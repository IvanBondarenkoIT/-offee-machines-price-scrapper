==============================================
НАСТРОЙКИ ПРОГРАММЫ
==============================================

Файл settings.ini содержит настройки программы.

ОСНОВНЫЕ ПАРАМЕТРЫ:
-------------------
[GENERAL]
- language: язык интерфейса (ru/en)
- create_pdf: создавать ли PDF отчет (yes/no)
- open_report: открывать ли отчет после создания (yes/no)

[URLS]
- Адреса сайтов для парсинга
- Можно изменить если сайты поменяли структуру

[TIMEOUTS]
- Настройки таймаутов
- Увеличьте если медленный интернет

[PARSING]
- Ожидаемое количество товаров
- Используется для проверки корректности парсинга

ВАЖНО:
------
1. НЕ изменяйте раздел [PATHS] без необходимости
2. Сохраняйте файл в кодировке UTF-8
3. После изменений перезапустите программу
