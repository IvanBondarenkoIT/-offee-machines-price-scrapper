==============================================
ПАПКА ДЛЯ ФАЙЛА ОСТАТКОВ
==============================================

ИНСТРУКЦИЯ:
-----------
1. Положите сюда файл с вашими остатками
2. Файл должен называться: остатки.xls или остатки.xlsx
3. Формат файла должен быть Excel

ФОРМАТ ФАЙЛА:
-------------
Файл должен содержать колонки:
- Наименование товара
- Количество
- Цена закупки

Пример:
| Наименование              | Кол-во | Цена   |
|---------------------------|--------|--------|
| DeLonghi EC9255.M         | 5      | 1500.00|
| DeLonghi ECAM22.110.B     | 3      | 950.00 |

ВАЖНО:
------
1. Товары DeLonghi будут извлечены автоматически
2. Остальные товары будут проигнорированы
3. Убедитесь что файл не поврежден
4. Закройте файл перед запуском программы

ПОСЛЕ ЗАПУСКА:
--------------
Программа автоматически найдет и обработает файл остатков.
Результаты будут в папке output/
