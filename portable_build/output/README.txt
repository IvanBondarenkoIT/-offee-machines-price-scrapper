==============================================
ПАПКА С РЕЗУЛЬТАТАМИ
==============================================

Здесь сохраняются результаты работы программы:

EXCEL ТАБЛИЦЫ (папка excel/):
-----------------------------
- price_comparison_YYYYMMDD_HHMMSS.xlsx
  Содержит:
  * Price Comparison - сравнение цен
  * Statistics - статистика
  * ALTA, KONTAKT, ELITE, DIM_KAVA - данные по каждому сайту
  * INVENTORY - ваши остатки

ОТЧЕТЫ (папка reports/):
------------------------
- executive_report_YYYYMMDD_HHMMSS.docx (Word)
- executive_report_YYYYMMDD_HHMMSS.pdf (PDF)
  
  Содержит:
  * Краткую сводку
  * Анализ конкурентоспособности
  * Возможности для прибыли
  * Рекомендации

ИСПОЛЬЗОВАНИЕ:
--------------
1. Excel таблица - для детального анализа
2. PDF отчет - для отправки директору
3. Файлы именуются по дате/времени создания

ХРАНЕНИЕ:
---------
- Файлы НЕ удаляются автоматически
- Можно сравнивать результаты за разные даты
- Рекомендуется периодически архивировать старые файлы
