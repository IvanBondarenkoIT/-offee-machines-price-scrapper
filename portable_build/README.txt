==============================================
PRICE MONITOR - PORTABLE VERSION
Система мониторинга цен DeLonghi
==============================================

БЫСТРЫЙ СТАРТ:
--------------
1. Положите файл остатков в папку inventory/
   Файл должен называться: остатки.xls
   
2. Запустите PriceMonitor.exe

3. Подождите 5-10 минут

4. Результаты в папке output/

СТРУКТУРА:
----------
PriceMonitor.exe       - Главная программа (запустить это!)
config/                - Настройки программы
  settings.ini         - Файл настроек
  README.txt           - Инструкция по настройке
inventory/             - Папка для файла остатков
  остатки.xls         - Положите сюда ваш файл
  README.txt           - Инструкция
output/                - Результаты работы
  excel/               - Excel таблицы сравнения
  reports/             - Word/PDF отчеты для директора
  README.txt           - Описание результатов
logs/                  - Логи работы программы

ТРЕБОВАНИЯ:
-----------
- Windows 10/11 (64-bit)
- 4 GB RAM
- Интернет соединение
- Chrome браузер (установится автоматически)

ЧТО ДЕЛАЕТ ПРОГРАММА:
---------------------
1. Парсит цены с сайтов конкурентов:
   - ALTA (alta.ge)
   - KONTAKT (kontakt.ge) 
   - ELITE (ee.ge)
   - DIM KAVA (dimkava.ge) - ваш сайт

2. Загружает ваши остатки из файла

3. Сопоставляет товары по моделям

4. Создает отчеты:
   - Excel таблица - детальное сравнение
   - PDF отчет - для директора

НАСТРОЙКА:
----------
Базовые настройки в файле config/settings.ini:
- URL сайтов для парсинга
- Таймауты (если медленный интернет)
- Создание PDF (yes/no)
- Автооткрытие отчета (yes/no)

ИСПОЛЬЗОВАНИЕ:
--------------
ЕЖЕДНЕВНО:
1. Обновить файл остатков
2. Запустить программу
3. Отправить PDF отчет директору

ЕЖЕНЕДЕЛЬНО:
1. Проанализировать Excel таблицы
2. Скорректировать цены
3. Архивировать старые отчеты

РЕШЕНИЕ ПРОБЛЕМ:
----------------
Проблема: "Файл остатков не найден"
→ Положите файл остатки.xls в папку inventory/

Проблема: "Ошибка парсинга"
→ Проверьте интернет соединение
→ Увеличьте таймауты в config/settings.ini

Проблема: "Chrome не найден"
→ Программа скачает его автоматически
→ Требуется интернет

Проблема: Программа долго работает
→ Нормально, парсинг занимает 5-10 минут
→ Смотрите логи в папке logs/
